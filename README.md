# Olist Marketplace Product Analytics

Портфолио-проект по Data Science / Machine Learning на основе датасета **Olist Brazilian E-Commerce Public Dataset** с Kaggle.

Проект создаётся для демонстрации практических навыков анализа данных, работы с несколькими связанными таблицами, построения бизнес-метрик, подготовки признаков и обучения ML-моделей.

## Цель проекта

Цель проекта — проанализировать данные маркетплейса, выявить ключевые бизнес-закономерности и построить модели машинного обучения, которые могут быть полезны для продуктовой и коммерческой аналитики.

В рамках проекта планируется показать навыки:

- загрузки и первичной проверки данных;
- очистки данных и анализа качества датасета;
- работы с несколькими связанными таблицами;
- exploratory data analysis;
- построения бизнес-метрик;
- визуализации данных;
- feature engineering;
- обучения и оценки ML-моделей;
- интерпретации результатов с точки зрения бизнеса.

## Датасет

Используется датасет **Olist Brazilian E-Commerce Public Dataset** с Kaggle.

Сырые данные не хранятся в репозитории, так как CSV-файлы добавлены в `.gitignore`.

Для воспроизведения проекта нужно скачать датасет с Kaggle и поместить CSV-файлы в папку:

```text
data/raw/
```

## План работы

1. Загрузить и изучить исходные таблицы.
2. Проверить качество данных: пропуски, дубликаты, типы данных.
3. Изучить связи между таблицами.
4. Провести exploratory data analysis.
5. Проанализировать заказы, клиентов, продавцов, товары, платежи и отзывы.
6. Построить ключевые бизнес-метрики.
7. Подготовить признаки для машинного обучения.
8. Обучить и сравнить несколько ML-моделей.
9. Интерпретировать результаты и сформулировать бизнес-выводы.
